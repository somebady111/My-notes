**1.标题栏——头**

```html
<!doctype html>
<html>
    <head>
    	<meta charset='utf-8'>
        <title>xxxxx</title>
    </head>
</html>
```



**2.内容——体**

​	在body中可以进行类型的添加，一般为设置容器（即div标签）。

​	列表标签：有序列表、无序列表

​	超链接标签：a

​	表格标签：table、tr一行、td表示单元格

​	表单标签：form

​	表单控件：input

```html
<!doctype html>
<html>
    <head>
        ...
    </head>
    <body>
        <div>
            <h1>
                
            </h1>
            <p>
                
            </p>
            <span></span>
            <label></label>
            <hr/>
            <br/>
            <ol>
                
            </ol>
            <ul>
                
            </ul>
            
        </div>
    </body>
</html>
```
